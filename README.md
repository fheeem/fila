FILA クローンプロジェクト

目的
本プロジェクトは、FILA公式サイトを参考に制作したクローンサイトです。
フロントエンド開発の基礎スキル向上を目的として、UIの再現およびレイアウト構造の理解に取り組みました。

実際のWebサイトを分析しながら、HTML / CSS / JavaScriptを用いて実装しています。

🛠 使用技術
HTML5
CSS3
JavaScript（Vanilla JS）

主な機能
メインページのレイアウト再現
ナビゲーションバーの実装
スクロールなど基本的なUI挙動の実装
一部レスポンシブ対応


本プロジェクトは学習目的で制作したクローンサイトであり、FILA公式とは関係ありません。
